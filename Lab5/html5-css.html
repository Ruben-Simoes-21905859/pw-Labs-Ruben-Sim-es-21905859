HTML5 e CSS

HTML 5

HTML5 é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML.


Elementos HTML5
header  Indicação dos cabeçalhos
nav  Indicação do menu de navegação com links para as páginas
main  Indicação dos conteúdos principais de cada página web
article  Indica porção do documento que pode ser vista de forma independente (não usado)
section  Especifica uma parte do conteúdo (não usado)
aside  Indica anotações (não usado)
footer  Indicação de rodapé da secção
details  Define detalhes adicionais que o utilizador pode ver ou esconder (não usado)
meter  Define uma medida escalar estática dentro de um intervalo (não usado)
progress  Define uma barra de progresso de uma tarefa (não usado)
abbr  Indica o significado de uma abreviatura (não usado)
address  Informação sobre o contacto do autor (não usado)

CSS

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags style. Também é possível, em vez de colocar a formatação dentro do documento, criar um link para um arquivo CSS que contém os estilos.


Seletores CSS
universal  Seleciona todos os elementos de todas as páginas, neste caso foi usado para mudar o tipo de letra.
tipo  Na selecção de todos os elementos de um tipo, usei para os headers 1, 2 e 3
classe  Quando classificamos com classe algum elemento com este tipo, podemos fazer alterações ao mesmo, ou seja foi isto que fiz
id  Quando classificamos com ID algum elemento com este tipo, podemos fazer alterações ao mesmo, ou seja foi isto que fiz x2
atributo  Este é quando queremos mudar algo afectado por determinada função, usei este para modificar uma hiperligação
pseudo-classe  Identificam elementos em determinado estado, na estrutura do documento, aqui terei deefinido o que acontece no menu quando cada botão é selecionado, visitado, passado por cima com o rato e quando este tá ativo
pseudo-elemento  Regras que se aplicam a partes de elementos, selecionei todas as primeiras letras de dento de vários "p´s" para ficarem com o tamnaho de 1.5


Seletores CSS
descendentes "espaço"  todos os p que estão “dentro” de um div
Filho ">"  todos os p que estão “logo dentro” de div
Adjacentes "~"  todos os p que estão colocados depois de um div
Imediatamente adjacentes "+"  todos os p que estão colocados “imediatamente” depois de um div
Agrupar ","  Agrupar os seletores,separando-os por virgulas
